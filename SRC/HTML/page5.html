Um accordion é um componente de interface de usuário que permite organizar e apresentar conteúdo de forma expansível e contraível. Geralmente, consiste em uma lista de itens onde apenas um item está expandido (mostrando seu conteúdo) de cada vez, enquanto os outros estão contraídos. Quando um item é clicado, ele se expande para mostrar seu conteúdo e os outros itens são contraídos automaticamente.
